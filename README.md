# Python_logic_exercices

Projet regroupant différents exercices de logique et d’algorithmique réalisés en Python.

## Petit projet Python pour pratiquer :

- Manipulation de chaînes (strings)

- Programmation orientée objet (OOP)

- Tableaux et structures de données (arrays/objects)

## Instalation

1. cloner le projet :

```bash
git clone XXX
cd python-logic-exercices
```

2. Installer avec Poerty les dépendances :

```bash
poetry install
```

3. Utilisation

Exécution du script principal main.py :

```bash
poetry run python -m python-logic-exercices.main
```

4. Tests

Exécuter les test unitaires avec pytest :

```bash
poetry run pytest
```
